# Opening Extension Instability

**Status:** Validated Phases 145, 146, 148  
**Confidence:** MEDIUM-HIGH

## Finding

**Opening extension narratives** (EARLY_EXTENSION, OPENING_DRIVE) frequently transition to EXHAUSTION or TRAP_REVERSAL — making high conviction at extension points unreliable.

## Evidence

| Module | Observation |
|--------|-------------|
| Phase 145 | `NARRATIVE_EXHAUSTED` trajectory elevated in opening sessions |
| Phase 145 transition failures | Extension → exhaustion breakpoints common |
| Phase 146 | Extension entries classified LATE/CHASE with poor expectancy |
| Phase 148 playbook profile | OPENING_EXTENSION → conviction DOWNGRADE required |

## Execution implication

- Downgrade conviction in opening extension without reclaim confirmation
- Require narrative stability before FULL_EXECUTION in first 15–30 minutes
- Phase 143: extension ≥ 7% → REDUCE_SIZE
